# FEP Symbol System

> **Grade 0 symbols.** Expanded system: [`via_resonantiae/symbols/SYMBOL_SYSTEM.md`](via_resonantiae/symbols/SYMBOL_SYSTEM.md)

> **Epistemic Status:** Constructed symbolism for contemplative use. Not evidence. Not traditional attribution.

---

## Primary Symbols

| Symbol | Name | Meaning |
|--------|------|---------|
| Well | Localization | Filter / boundary that holds excitation |
| Wave | Excitation | You as mode in the field |
| Twin Waves | Coupling | Resonant dyad; ethical synchrony |
| Diffuse Cloud | Bleed Warning | Filter failure; stop practice |

---

## Usage

- Draw or visualize the **Well** before imaginal work (boundary setting)
- **Wave** during solo mode exploration
- **Twin Waves** during dyad coupling ritual
- **Diffuse Cloud** as stop signal if session feels disorganized

---

## What We Do Not Use

- Traditional Golden Dawn tarot attributions as proof
- Thelemic god-forms as entities
- Planetary seals as literal keys to planes

Historical esotericism informs vocabulary; FEP symbols are **new construction**.

See [`historical_lineage.md`](historical_lineage.md).
