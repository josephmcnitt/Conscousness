# Grade System — Via Resonantiae

> **Epistemic Status:** Constructed progression. Not secret initiation lineage.

---

## Overview

| Grade | Name | Minimum time | Primary way |
|-------|------|--------------|-------------|
| **0** | Spark | 2–4 weeks | Logos + Pathos |
| **I** | Threshold | 4–6 weeks | Pathos |
| **II** | Modes | 4–6 weeks | Logos + Pathos |
| **III** | Resonance | 6+ weeks | Pistis + Pathos |
| **IV** | Integration | 6+ weeks | Pistis |
| **V** | Silent Gate | Optional | Gnosis |

Advance when progression criteria met (below) and no active STOP_CONDITIONS.
